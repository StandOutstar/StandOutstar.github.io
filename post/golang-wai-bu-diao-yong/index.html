Golang-外部调用

2019-08-21

调用外部命令

  • 生成命令
  • 设置命令输出目标
  • 运行命令
  • 运行并获得命令输出
  • 管道连接
  • 设置命令环境变量
  • 查找命令路径

生成命令

  • cmd := exec.Command("Program", "arg1", "arg2", ...)
  • cmd := exec.CommandContext(context, "Program", "arg1", "arg2", ...)，可以设置 context

设置命令输出目标

  • 设置命令标准输出
cmd.Stdout = os.Stdout
  • 设置命令标准错误输出
cmd.Stderr = os.Stderr

运行命令

  • 同步方式：
cmd.Run()
  • 异步方式
cmd.Start() 
cmd.Wait() // 等待命令执行完成，并关闭自生成管道

运行并获得命令输出

  • cmd.Output()， 只有stdout输出
  • cmd.CombinedOutput()，stdout 和 stderr 输出

管道连接

  • 自定义管道
r, w := io.Pipe() // 自定义管道
cmd1.Stdout = w
cmd2.Stdin = r
  • cmd 生成管道
cmd1Out, _ := cmd1.StdoutPipe() 
cmd2.Stdin = cmd1Out 

// or 
cmd2In, _ := cmd2.StdinPipe() 
cmd1.Stdout = cmd2In

设置命令环境变量

cmd.Env = newEnv

查找命令路径

path, err := exec.LookPath("ls")
if err != nil {
    fmt.Printf("didn't find 'ls' executable\\n")
} else {
    fmt.Printf("'ls' executable is in '%s'\\n", path)
}
